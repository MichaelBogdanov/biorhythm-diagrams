# Приложение на C# для построения диаграмм биоритмов человека по дате рождения
Делал данное небольшое приложение в рамках учебного проекта, когда учился в колледже

> Разработать приложение «Биоритмы человека» с использованием компонентов DateTimePicker, Chart и других элементов интерфейса.
Приложение должно строить биоритмы человека по дате его рождения на исследуемый период. 
На основе анализа индивидуальных биоритмов необходимо прогнозировать неблагоприятные дни, 
выбрать благоприятные дни для разного рода деятельности. 
Существует гипотеза, что жизнь человека подчиняется трем циклическим процессам, называемых биоритмами: 
физическому (продолжительность 23 дня), эмоциональному (продолжительность 28 дней) и интеллектуальному (продолжительность 33 дня). 
Биоритмы характеризуют подъемы и спады нашего состояния в виде синусоиды. 
За точку отсчета всех трех биоритмов берется день рождения человека, для которого он строится.
